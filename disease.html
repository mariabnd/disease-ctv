CRAN Task View: Disease

Maintainer:mariabnd
Version:2020-02-01
URL:https://CRAN.R-project.org/view=Disease

This Task View contains a list of packages useful for analysis of disease data. The name is chosen such that it covers both infectious disease epidemiology and disease ecology. NB this is not an official CRAN Task View so the link above will not work.

If you would like to suggest additions to this list, please create a GitHub pull request noting the package(s) you'd like to suggest for inclusion with a short description of the features provided as well as a link to the package
